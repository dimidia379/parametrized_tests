Тест-кейс №1

Заголовок: "Отображение сообщения об ошибке при вводе мейла без символа "@" при регистрации на pinterest.ru"
Приоритет: BLOCKER
Предусловия: Открыт сайт https://www.pinterest.ru/

Шаги:
1) Нажать кнопку "Sign up"
2) В поле "email" ввести мейл без символа "@":  test.mail.com
3) Нажать кнопку "Continue"

Ожидаемый результат:
- Под полем "email" отображается сообщение об ошибке с текстом "Hmm...that doesn't look like an email address"
